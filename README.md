# Craft CMS v2 Boilerplate

### Includes:

* gulpfile.js set-up
* Basic SASS structure
* node_modules
* package.json
* JS vendors
* CSS vendors

## Gulp Packages included:

* gulp
* gulp-sass
* gulp-sourcemaps
* gulp-concat
* gulp-rename
* gulp-uglify
* gulp-clean-css
* gulp-order
* gulp-autoprefixer

## JS files:

* public/assets/js/forms.js (ajax form validation and handler using Malsup jQuery forms and jQuery validate)
* public/assets/js/custom.js (global JS)

## JS vendors:

* jQuery Validate
* Additional Methods
* jQuery Forms Malsup
* Equal Height (soon to be deprecated in favour of flexbox)
* Owl Carousel v2.2.1
* ScrollMe
* Stacktable
* FitVid
* Fancybox v3
